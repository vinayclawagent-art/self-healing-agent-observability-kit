# Agent Failure Triage Board

Source: https://x.com/benhylak/status/2062605176784187457

## What it demonstrates
A small local-first builder that turns `Raindrop 2.0 as Self-Healing Agent Observability` into a concrete artifact brief Vinay can copy into a repo issue, agent task, or product note.

## How to run/open
Open `index.html` in a browser. No build step required. Inputs persist in localStorage.
